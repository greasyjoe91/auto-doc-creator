📦 部署包内容：
  ├─ 一键部署环境.bat  # 自动安装环境和启动服务
  ├─ stop.bat          # 停止服务
  ├─ 使用说明.md       # 详细使用文档
  ├─ dist/             # 前端构建产物
  └─ server/           # 后端服务代码

🚀 快速开始：
  1. 双击运行 "一键部署环境.bat"
  2. 脚本会自动检测并安装 Node.js（如需要）
  3. 自动安装项目依赖
  4. 自动启动服务
  5. 浏览器自动打开 http://localhost:3005

⚙️ 首次使用配置：
  编辑 server\.env 文件，配置 Gemini API Key：
  GEMINI_API_KEY=your_api_key

  获取 API Key: https://aistudio.google.com/apikey

📝 注意事项：
  - 需要网络连接（用于下载 Node.js 和依赖）
  - 首次安装 Node.js 后需重启脚本
  - 确保 3002 端口未被占用
  - 查看详细说明请阅读 "使用说明.md"

🛠️ 常用命令：
  启动服务: 双击 "一键部署环境.bat"
  停止服务: 双击 "stop.bat"
  查看日志: 打开 server.log 文件

📞 技术支持：
  如遇问题，请查看 "使用说明.md" 中的常见问题部分
