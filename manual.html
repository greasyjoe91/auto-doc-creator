AutoDoc Flow

智能化 SOP 文档生成系统操作手册

1. 软件简介

AutoDoc Flow 是一款基于 Google Gemini 3 大模型的智能化文档生成工具。它能帮助用户将原始的 SOP（标准作业程序）、CSV 数据或简单的操作脚本，一键转化为专业级的技术规格书、营销文案和用户操作手册。

核心亮点

  • 多模态输入：支持 PDF, Word, Excel, CSV 及视频文件 (MP4)。
  • 视频智能截图：自动从录屏中提取高清关键帧，并自动匹配插入到文档对应位置。
  • 模版克隆：上传已有文档作为模版，AI 将严格模仿其格式和文风。

2. 界面概览

软件界面分为左侧“输入控制区”和右侧“文档生成预览区”。

左侧栏 (输入)

  • • SOP 输入数据：核心数据源，支持文本粘贴或文件上传。
  • • 软件录屏 / 配图：上传视频，AI 自动提取素材。
  • • 参考模版：上传旧文档，让 AI 学习风格。

右侧栏 (输出)

  • • 文档选项卡：切换规格书、营销文案、操作手册。
  • • 操作栏：生成、复制、导出 HTML。
  • • 预览区：实时渲染 Markdown 内容和图片。

3. 操作流程

1 导入 SOP 数据

这是生成所有文档的基础。您可以直接粘贴文本，也可以上传文件。

  • 支持 Google Sheet 链接（需开启共享权限）。
  • 支持 PDF, Word, Excel, CSV 文件。
[请在此处粘贴：左上角 SOP 输入区域截图，展示上传了一个 Excel 文件的状态]

2 上传录屏视频 (可选)

如果您希望生成的《产品介绍》或《用户手册》包含配图，请在此处上传软件操作录屏 (MP4)。

注意： 系统会自动提取约 20 张高清关键帧作为“素材库”。AI 会根据文案内容，自动从中挑选最合适的图片插入。
[请在此处粘贴：左侧中间视频上传区域截图，展示已提取出缩略图的状态]

3 配置参考模版 (可选)

如果您有固定的公司文档格式，请在左下角上传对应的参考文件（如 PDF 或 Word）。AI 会“克隆”其排版风格。

[请在此处粘贴：左下角参考模版区域截图]

4 生成文档

在右侧顶部切换您需要生成的文档类型，然后点击蓝色 “开始生成” 按钮。

推荐生成顺序：

  1. 先生成 1. 软件参数规格书 (分析基础数据，不含图)。
  2. 再生成 2. 软件产品介绍 (基于规格书，自动配图)。
  3. 最后生成 3. 软件用户操作手册 (基于规格书+SOP，自动配图)。
[请在此处粘贴：右侧生成中文档的加载状态，或生成完成后的预览截图]

5 导出与分享

文档生成满意后，您可以使用右上角的工具栏：

  • 复制：将内容（包含图片）复制到剪贴板，可直接粘贴到 Google Docs 或 Word。
  • 导出：下载为 HTML 格式文件，方便离线查看。
